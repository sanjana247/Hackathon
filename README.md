# Cybercrime NLP Model
This project develops an NLP model to assist citizens in filing cybercrime reports correctly on the NCRP.

## Project Structure
- `data/`: Dataset files.
- `notebooks/`: Jupyter notebooks for exploration.
- `src/`: Scripts for preprocessing, model building, and evaluation.
- `app/`: Flask app for deploying the model.
- `README.md`: Project documentation.
